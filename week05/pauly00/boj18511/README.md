## 문제

[문제 링크](https://boj.kr/18511)

<p><em>N</em>보다 작거나 같은 자연수 중에서, 집합&nbsp;<em>K</em>의 원소로만 구성된 가장 큰 수를 출력하는 프로그램을 작성하시오. <em>K</em>의 모든 원소는 1부터 9까지의 자연수로만 구성된다.</p>

<p>예를 들어 <em>N</em>=657이고, <em>K</em>={1, 5, 7}일 때 답은 577이다.</p>

## 입력

<p>첫째 줄에 <em>N</em>, <em>K</em>의 원소의 개수가 공백을 기준으로 구분되어 자연수로 주어진다. (10 ≤&nbsp;<em>N&nbsp;</em>≤ 100,000,000, 1 ≤&nbsp;<em>K</em>의 원소의 개수 ≤ 3) 둘째 줄에 <em>K</em>의 원소들이 공백을 기준으로 구분되어 주어진다. 각 원소는 1부터 9까지의 자연수다.</p>

<p>단, 항상 <em>K</em>의 원소로만 구성된&nbsp;<em>N</em>보다 작거나 같은 자연수를 만들 수 있는 경우만 입력으로 주어진다.</p>

## 출력

<p>첫째 줄에 <em>N</em>보다 작거나 같은 자연수 중에서, <em>K</em>의 원소로만 구성된 가장 큰 수를 출력한다.</p>
