### 📂 Repository 구조

스터디는 **`주차/이름/문제.언어`** 구조로 관리합니다.

매주 해당 주차 폴더 안에 **본인의 이름(ID)으로 된 폴더**를 생성해주세요.

```text
Algorithm-Study/
├── week01/
│   ├── noiskk/                <-- 본인 이름 폴더 생성
│   │   ├── boj_1000.py        <-- 문제 소스코드
│   │   └── boj_1001.py
│   └── member1/
│       ├── boj_1000.java
│       └── boj_1001.java
├── week02/
│   ├── noiskk/
│   └── ...
└── ...

```
